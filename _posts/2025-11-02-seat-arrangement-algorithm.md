한 달에 한 번, 교실에서는 보이지 않는 전쟁이 일어납니다. 바로 '자리 바꾸기' 시간입니다. 제비뽑기는 공정해 보이지만, 시력이 나쁜 학생, 키가 큰 학생, 혹은 사이가 좋지 않은 학생들을 고려하지 못한다는 단점이 있습니다. 그래서 저는 **제약 조건을 만족하는 자리 배치 알고리즘**을 직접 만들어보기로 했습니다.

![코딩 화면](https://images.unsplash.com/photo-1515879218367-8466d910aaa4?ixlib=rb-4.0.3&auto=format&fit=crop&w=1200&q=80)

## 1. 요구사항 분석

자리 배치 프로그램이 갖춰야 할 조건은 다음과 같았습니다.

- **필수 조건:** 시력이 나쁜 학생은 앞자리에 배정한다.
- **필수 조건:** 남녀 짝꿍을 기본으로 하되, 성비가 맞지 않으면 일부는 동성 짝꿍을 허용한다.
- **선택 조건:** 지난달 짝꿍과는 다시 짝이 되지 않도록 한다.
- **선택 조건:** 다툼이 있었던 학생들은 서로 멀리 떨어뜨린다.

## 2. 알고리즘 구현

Python을 사용하여 백트래킹(Backtracking) 기법과 유전 알고리즘(Genetic Algorithm)을 혼합하여 구현했습니다. 모든 경우의 수를 다 따지기에는 연산량이 너무 많기 때문에, 제약 조건을 만족하는 해를 효율적으로 찾기 위함입니다.

```python
def check_constraints(seat_map, student):
    # 시력 조건 확인
    if student.vision < 0.5 and seat_map.row > 1:
        return False
    # 기피 학생 거리 확인
    for other in seat_map.neighbors:
        if other in student.avoid_list:
            return False
    return True
```

## 3. 적용 결과

프로그램을 돌려 나온 결과를 화면에 띄워주자 아이들의 반응이 뜨거웠습니다. "와, 선생님 이거 AI가 짠 거예요?"라며 신기해했습니다.

무엇보다 불만이 크게 줄었습니다. "왜 제가 얘랑 짝이에요?"라는 불평 대신, "아, 조건 때문에 이렇게 됐구나"하고 납득하는 분위기가 형성되었습니다. 알고리즘이 가져다준 공정함이 교실의 평화를 지켜준 셈입니다.
