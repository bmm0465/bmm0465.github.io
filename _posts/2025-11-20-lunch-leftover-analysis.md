초등 교실에서 점심 시간은 단순한 식사 시간을 넘어, 생활 지도와 식습관 교육이 이루어지는 중요한 시간입니다. 하지만 "골고루 먹어라", "남기지 마라"는 잔소리는 아이들에게 잘 와닿지 않습니다. 그래서 저는 **데이터**를 활용해 보기로 했습니다.

![데이터 시각화 예시](https://images.unsplash.com/photo-1551288049-bebda4e38f71?ixlib=rb-4.0.3&auto=format&fit=crop&w=1200&q=80)

## 1. 문제 정의

우리 반 학생들은 유독 채소 반찬을 많이 남기는 경향이 있었습니다. 매일 발생하는 잔반의 양을 직관적으로 보여주고, 이를 줄이기 위한 동기 부여가 필요했습니다.

## 2. 데이터 수집

데이터 수집은 아주 아날로그적인 방식에서 시작했습니다. 급식 지도 도우미 학생들이 식판을 검사하며 잔반의 양을 5단계 척도로 기록했습니다.

- **5점:** 싹싹 다 먹음 (국물 조금 남김)
- **4점:** 반찬 1개 조금 남김
- **3점:** 밥이나 반찬을 절반 정도 남김
- **2점:** 많이 남김
- **1점:** 거의 먹지 않음

## 3. 데이터 분석 및 시각화

수집된 데이터를 엑셀과 Python의 `matplotlib` 라이브러리를 활용하여 시각화했습니다. 요일별 잔반량, 메뉴별 선호도 등을 그래프로 그려 교실 앞 TV에 게시했습니다.

```python
import matplotlib.pyplot as plt
import pandas as pd

# 데이터 로드
data = pd.read_csv('lunch_data.csv')

# 요일별 잔반 평균 시각화
plt.figure(figsize=(10, 6))
plt.bar(data['Day'], data['Leftover_Score'], color='#3b82f6')
plt.title('요일별 평균 잔반 점수')
plt.show()
```

> "선생님, 우리가 화요일에 제일 많이 남기네요? 맛없는 메뉴가 나와서 그런가 봐요."

아이들은 그래프를 보며 스스로 원인을 분석하기 시작했습니다. 단순히 혼나는 것이 아니라, **현상을 객관적으로 인지**하게 된 것입니다.

## 4. 변화와 결론

데이터를 공유한 지 2주 후, 잔반 없는 날(Zero Waste Day) 챌린지를 진행했을 때 성공률이 이전보다 40% 이상 증가했습니다. 데이터는 아이들에게 잔소리가 아닌 **피드백**이 되었고, 스스로 행동을 수정하는 계기가 되었습니다.

이처럼 교실 속 작은 데이터들도 교육적으로 활용하면 큰 변화를 이끌어낼 수 있습니다.
